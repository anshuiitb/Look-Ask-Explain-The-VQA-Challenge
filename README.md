## Week 2: Natural Language Processing
### Projects Implemented

#### 1. BERT-based Sentiment Analysis

- **Technologies**: Transformers, PyTorch, BERT
- **Key Features**:
  - Pre-trained BERT model fine-tuning
  - Sentiment classification (positive/negative)
  - Model evaluation and performance metrics

#### 2. SMS Spam Classification
- **Objective**: Classify SMS messages as spam or legitimate
- **Dataset**: SMSSpamCollection dataset
- **Approach**: Text preprocessing and classification techniques

```

git clone https://github.com/yourusername/Ask-the-image.git
cd Ask-the-image

python -m venv venv

pip install -r requirements.txt
```
